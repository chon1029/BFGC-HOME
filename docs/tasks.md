# 📝 BFGC 프로젝트 작업 리스트 (Backlog)

이 문서는 프로젝트 진행 중 "나중에 하기로 미뤄둔 작업"들을 기록하여, 누락 없이 완성도 높은 결과물을 만들기 위한 리스트입니다.

## 🚀 우선순위 높음 (High Priority)

### 1. 데이터베이스 (Sanity) 연동
- [ ] **설교(Sermon) 데이터 연동:** 현재 Mock 데이터를 Sanity API(`createClient`)를 통해 실제 데이터로 교체.
- [ ] **회원(User) 데이터 연동:** NextAuth 회원가입 시 Sanity `user` 스키마에 저장하도록 로직 구현.
- [ ] **오늘의 양식(Daily Bread) 데이터 연동:** 스키마 생성 및 페이지 연동.
- [ ] **제자훈련(Discipleship) 데이터 연동:** 스키마 생성 및 페이지 연동.

### 2. 관리자 페이지 (Admin)
- [ ] **실제 데이터 통계:** 대시보드의 방문자 수, 가입자 수 등을 실제 DB 카운트로 변경.
- [ ] **회원 관리 기능:** 회원 목록 조회, 등급 변경(관리자 승격 등), 강제 탈퇴 기능 구현.
- [ ] **스팸 차단 기능:** 실제 IP 차단 로직 또는 스팸 필터링 구현.
- [ ] **Sanity Studio 바로가기:** 관리자 사이드바에 Studio로 이동하는 직관적인 버튼 추가.

### 3. 인증 시스템 (Auth)
- [ ] **이메일 발송 시스템:** `Resend` 또는 `Nodemailer` 연동하여 비밀번호 재설정 메일 실제 발송.
- [ ] **소셜 로그인 확장:** 카카오톡, 네이버 로그인 추가 (한국 사용자 편의성 증대).
- [ ] **프로필 이미지 업로드:** Sanity에 프로필 사진 업로드 기능 구현.
